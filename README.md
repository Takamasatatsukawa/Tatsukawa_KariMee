# 開発言語
 *　Ruby 3.3.0
 *  Rails 7.1.3.4

# 実行手順
$ git clone git@github.com:Takamasatatsukawa/Tatsukawa_KariMee.git
$ cd KariMeeApp
$ bundle install
$ rails db:create && rails db:migrate && rails db:seed
$ rails s
